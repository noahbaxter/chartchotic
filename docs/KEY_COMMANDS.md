# Key Commands

All keyboard and mouse shortcuts in Chartchotic. Commands are grouped by when they're available.

## Global

| Key | Action |
|-----|--------|
| W | Toggle authoring mode |

## Scroll Wheel

| Modifier | Action |
|----------|--------|
| None | Scroll timeline (2 beats per tick) |
| Shift | Scroll timeline fine (0.5 beats per tick) |
| Cmd | Adjust note speed |

## Authoring Mode

| Key | Action |
|-----|--------|
| Q | Toggle Draw / Edit sub-mode |
| S | Toggle snap to grid |
| T | Cycle tuplet (off → 3 → 5 → 7 → off) |
| [ | Step division down |
| ] | Step division up |
| B | Toggle bar mode |
| Delete / Backspace | Delete selected notes |
| Escape | Deselect all (or clear stamp if active) |

## Draw Mode
### Mouse

| Input | Action |
|-------|--------|
| Left click | Place note |
| Left drag | Place note + sustain drag |
| Shift + left click | Paint mode (continuous placement) |
| Shift + left drag | Paint mode drag |
| Right click | Erase note under cursor (clears stamp if active) |
| Right drag | Erase marquee |

### Stamp Tool

| Input | Action |
|-------|--------|
| C (in edit mode, 2+ notes selected) | Capture selection as stamp, switch to draw |
| Hold C (in draw mode) + move mouse | Marquee stamp capture (commits on C release) |
| Left click (while stamp active) | Place stamp |
| Left drag (while stamp active) | Place stamp + extend sustains |
| Shift + left drag (while stamp active) | Paint stamp |
| Left/Right arrows (while stamp active) | Shift stamp lanes |
| Escape (while stamp active) | Clear stamp |
| Right click (while stamp active) | Clear stamp + enter erase |

## Edit Mode
### Mouse

| Input | Action |
|-------|--------|
| Left click on note | Select note |
| Left click on empty | Begin marquee select |
| Left drag | Marquee select or move (context-dependent) |
| Alt + left drag | Axis-locked move |
| Double-click on note | Delete note |
| Double-click on empty | Create note |

### Keyboard

| Key | Action |
|-----|--------|
| Up arrow | Move selection forward in time (by current step) |
| Down arrow | Move selection backward in time (by current step) |
| Left arrow | Move selection left one lane |
| Right arrow | Move selection right one lane |
| Delete / Backspace | Delete selection |
| Escape | Deselect all |
